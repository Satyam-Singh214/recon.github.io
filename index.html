Subdomain Enumeration

Discover subdomains using passive and active techniques and detect takeover opportunities.

Passive Subdomains

Subfinder

Collect subdomains using passive sources only.

subfinder -d example.com -silent

Active (Permutation) Subdomains

puredns

Bruteforce permutations using a large wordlist.

puredns bruteforce -d example.com -w /wordlists/all.txt

Subdomains & Takeover Check

subfinder + subjack

Find subdomains and check for potential takeovers.

subfinder -d example.com | subjack -w fingerprint.txt -t 100

ASN & IP Discovery

Map organizational ownership and resolve domains to IPs.

ASN & IP Info (Whois)

whois

Get ASN, CIDR and organization information.

whois example.com | grep -E "CIDR|AS Name|OrgName"

Resolve to IP

resolveip

Resolve domain to IP address.

echo example.com | resolveip

Live Host Discovery

Identify live URLs and IPs to focus your attacks.

Probe for Live URLs

httpx

Probe subdomains for live web services.

cat subdomains.txt | httpx -silent -title -webserver

Live IPs (Scan)

nmap

Ping sweep over a list of IPs.

nmap -sP -iL ip_list.txt

URL Collection & Analysis

Harvest historical URLs, parameterized endpoints and JS‑exposed paths.

From Wayback Machine

waybackurls

Grab archived URLs and filter non‑static assets.

waybackurls example.com | grep -v "jpg\|png\|gif" | sort -u

URLs with Parameters

grep

Extract URLs containing parameters for further testing.

cat urls.txt | grep "=" | sort -u

Endpoints from JS Files

getallurls

Pull endpoints leaked inside JavaScript files.

subfinder -d example.com | getallurls -js | sort -u

Vulnerability Scanning

Quick, low‑noise vulnerability checks for common issues.

Low‑Severity Scan

nuclei

Run nuclei templates for default logins and misconfigurations.

nuclei -l urls.txt -t default-logins.yaml

LFI Payload Fuzzing

ffuf

Fuzz file inclusion parameters using LFI payload list.

ffuf -w LFI_payloads.txt -u example.com/file?id=FUZZ

Sensitive File Discovery

Locate robots, backups and exposed configuration files.

Robots.txt Analysis

curl + grep

Extract disallowed paths and sitemaps.

curl -s example.com/robots.txt | grep -E "Disallow:|Sitemap:"

Common Backups

ffuf

Bruteforce common backup filenames.

ffuf -w wordlists/common_backups.txt -u example.com/FUZZ

Hidden Parameter Discovery

Uncover hidden HTTP parameters for further fuzzing and logic testing.

Param Mining (Basic)

Arjun

Automatically discover parameters for a given endpoint.

Arjun -u example.com/index.php

Guess Parameters

ffuf

Guess parameters with a custom wordlist.

ffuf -w wordlists/params.txt -u example.com/index.php?FUZZ=1

Directory & File Bruteforcing

Enumerate hidden directories, files and interesting extensions.

Dir & File Discovery

gobuster

Bruteforce directories and files with wordlists.

gobuster dir -u example.com -w /wordlists/dir.txt -t 50

Extension Enumeration

ffuf

Probe for alternate extensions of files.

ffuf -w /wordlists/extensions.txt -u example.com/indexFUZZ

WordPress Security Testing

Enumerate versions, plugins and themes for vulnerable components.

WP Version & Info

wpscan

Enumerate core version and vulnerabilities.

wpscan --url example.com --enumerate vp

Plugin & Theme Scan

wpscan

Scan for vulnerable plugins and themes.

wpscan --url example.com --enumerate ap,at

CORS Testing

Detect misconfigured Cross‑Origin Resource Sharing policies.

CORS Misconfig Check

nuclei

Bulk test URLs for CORS misconfigurations.

cat urls.txt | nuclei -t cors-misconfig.yaml

Manual Origin Fuzz

curl

Send a crafted Origin header to probe CORS behaviour.

curl -i -X GET "https://example.com" -H "Origin: evil.com"
